Tools
=====

Various tools are available in layered packages such as mdbm_tools
and mdbm_stat_monitor.

.. toctree::
   :maxdepth: 1

   mdbm_get_config - Shows MDBM configuration information <mdbm_get_config>
   mdbm_stat_monitor.pl - Monitor mdbm_stat output <mdbm_stat_monitor>
   mdbm_big_data_builder.pl - Fast building for a larger-than-physical-memory MDBM <mdbm_big_data_builder>
   mdbm_export_splitter - Splits the provided key-value data source files into buckets <mdbm_export_splitter>
   mdbm_config - Optimize dataset-specific MDBM parameters for improved performance <mdbm_config>
